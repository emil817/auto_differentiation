# Автоматическое символьное дифференцирование

Поддерживаются Latex-выражения.

[Пример работы](example.py)

### Функционал

Взятие производной по переменной заданного порядка:
```
find_deriv(formula: str, var: str = 'x', order: int = 1)
```

Взятие производной по заданной переменной:
```
der_by_var(formula: str, var: str)
```

Взятие производной по любой переменной:
```
der(formula: str)
```

Поиск градиента функции:
```
find_gradient(formula: str)
```
